### Day9 特製雕花 Vue-cli 3.0 與 Node.js 開發環境

#### 內容簡介

簡單介紹Vue CLI與安裝環境

#### 重點整理
- 安裝vue cli
- vue cli 介面介紹
- vue cli 功能 
``` 
1. server 開啟server
2. build 將專案包起來(編譯並壓縮)可給後端執行
3. lint 看編碼規則是否一致
4. inspect 檢查webpack配置
```

Code 
安裝vue cli
```javascript
npm install -g @vue/cli
# OR
yarn global add @vue/cli
```
創建項目
```javascript
vue create my-project
# OR
vue ui
```
